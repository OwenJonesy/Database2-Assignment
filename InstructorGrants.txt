Assessment Table:

SELECT privilege to read assessments.
INSERT privilege if you need to insert new assessments.
UPDATE privilege if you need to update existing assessments.
You can grant these privileges using SQL statements like:

GRANT SELECT ON Assessment TO instructor_role;
GRANT INSERT, UPDATE ON Assessment TO instructor_role;
